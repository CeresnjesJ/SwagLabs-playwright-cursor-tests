# SwagLabs-playwright-cursor-tests
End-to-end automated tests for SauceDemo.com using Playwright, TypeScript, and Page Object Model. AI-assisted test development with Cursor.
## Quick Start
```bash
npm install
npx playwright install
npx playwright test
npx playwright show-report
```

## Structure
```
tests/       # Test files (login, home, cart, checkout)
pages/       # Page Object Model classes
helpers/     # Utility functions
```

## Coverage

- Authentication & session management
- Product browsing & sorting
- Shopping cart operations
- Complete checkout flow

## Tech Stack

- Playwright
- TypeScript
- Page Object Model
- Cursor AI assistance
